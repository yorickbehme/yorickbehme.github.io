KI-Testgenerierungs Tool (2024/2025)

  • Im Rahmen meiner Bachelorarbeit habe ich ein Tool entwickelt, welches die lokale KI-Generierung von Testskripten für die ArTEMiS-Lernplattform vereinfacht. Das Python Tool sollte dabei leicht erweitert werden können und möglichst einfach zu bedienen sein.

    Repository

Blindenhund Roboter App (2024)

  • Dieses Projekt habe ich im Rahmen meiner Forschungsarbeit an der Zhejiang-Universität entwickelt. Mit der prototypischen App lässt sich ein Deep Robotics Lite 2 Roboter steuern. Eine besondere Herausforderung war es die App so zu gestalten, dass sie von Menschen mit Sehbehinderung genutzt werden kann. Die App wurde mit Java für das Android Betriebssystem entwickelt.

    Repository

Smartwatch AR Erweiterung (2023/2024)

  • In diesem Projekt habe ich gemeinsam mit zwei Kommilitonen einen Prototypen für eine AR Erweiterte Smartwatch mit taktilem Armband entwickelt. Die Entwicklung erfolgte in Unity mit C# und in Tizen Studio mit Webtechnologien.

Lernspiel Hardwarekomponenten (2023/2024)

  • Gemeinsam im Team haben wir mit Godot ein Lernspiel zum Thema Hardwarekomponenten für SchülerInnen entwickelt. Das besondere war die Zielplattform eines Screentables.

Hotelmanagement-Website (2022/2023)

  • In diesem Projekt haben wir gemeinsam als Team prototypisch eine Website für ein Hotelmanagement entwickelt. Die Seite beinhaltet typische Funktionen wie Buchung, Bewertung und Verwaltung von Zimmern. Der Fokus in dem Projekt lag auf der Backend Entwicklung mit Java und Spring.

Roboter Steuerung (2021)
